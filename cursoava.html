¿Qué es CSS Avanzado?

CSS (Cascading Style Sheets) avanzado permite crear diseños web complejos y atractivos mediante técnicas avanzadas como flexbox, grid, animaciones y transiciones, y preprocesadores como SASS.

Flexbox

Flexbox es un modelo de diseño unidimensional que permite distribuir el espacio dentro de un contenedor y alinear los elementos internos de manera flexible.

Ejemplo:

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

.item {
    background-color: #3498db;
    padding: 20px;
    margin: 10px;
    color: white;
}

Grid

CSS Grid Layout es un sistema de diseño bidimensional que permite dividir una página en regiones o definir la relación de tamaño, posición y capas entre partes del control construido a partir de primitivas del HTML.

Ejemplo:

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

.item {
    background-color: #2ecc71;
    padding: 20px;
    color: white;
}

Animaciones y Transiciones

Las animaciones y transiciones permiten agregar interactividad y dinamismo a las páginas web. Las transiciones cambian gradualmente los valores de las propiedades CSS, mientras que las animaciones permiten definir movimientos más complejos.

Ejemplo:

.box {
    width: 100px;
    height: 100px;
    background-color: #e74c3c;
    transition: transform 0.5s;
}

.box:hover {
    transform: rotate(45deg);
}

Preprocesadores CSS (SASS)

SASS (Syntactically Awesome Style Sheets) es un preprocesador CSS que permite usar variables, anidación, mixins, herencia y otras características que hacen que CSS sea más poderoso y fácil de mantener.

Ejemplo:

$primary-color: #3498db;

.container {
    .item {
        color: $primary-color;
        @include border-radius(5px);
    }
}
