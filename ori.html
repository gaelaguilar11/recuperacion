Programación Orientada a Objetos (POO)

Introducción a la Programación Orientada a Objetos (POO)

La programación orientada a objetos (POO) es un paradigma de programación basado en el concepto de "objetos", que pueden contener datos y código: datos en forma de campos, y código en forma de procedimientos conocidos como métodos. La POO se utiliza para modelar objetos del mundo real y sus interacciones.

Principios de la POO

Encapsulamiento

El encapsulamiento es la ocultación de los detalles de implementación y el acceso a los datos solo a través de métodos definidos. Esto ayuda a proteger los datos y a modular el código.

Abstracción

La abstracción se refiere a la simplificación de la complejidad mediante la modelización de clases apropiadas según el contexto. Permite enfocarse en los aspectos esenciales y ocultar los detalles irrelevantes.

Herencia

La herencia es un mecanismo mediante el cual una clase puede heredar propiedades y métodos de otra clase. Esto promueve la reutilización de código y establece una relación jerárquica entre clases.

Polimorfismo

El polimorfismo permite que diferentes clases respondan de manera diferente a la misma llamada de método. Esto se logra mediante la sobrescritura de métodos en clases derivadas.

Ejemplos de POO en JavaScript

Creación de una Clase

class Persona {
    constructor(nombre, edad) {
        this.nombre = nombre;
        this.edad = edad;
    }

    saludar() {
        return `Hola, me llamo ${this.nombre} y tengo ${this.edad} años.`;
    }
}

// Crear una instancia de la clase Persona
const persona1 = new Persona('Juan', 25);
console.log(persona1.saludar());

En este ejemplo, se crea una clase Persona con un constructor que inicializa los atributos nombre y edad. Además, se define un método saludar que devuelve un saludo con el nombre y la edad de la persona.

Herencia

class Estudiante extends Persona {
    constructor(nombre, edad, carrera) {
        super(nombre, edad);
        this.carrera = carrera;
    }

    estudiar() {
        return `${this.nombre} está estudiando ${this.carrera}.`;
    }
}

// Crear una instancia de la clase Estudiante
const estudiante1 = new Estudiante('Ana', 22, 'Ingeniería');
console.log(estudiante1.saludar());
console.log(estudiante1.estudiar());

En este ejemplo, se crea una clase Estudiante que hereda de la clase Persona. La clase Estudiante añade un nuevo atributo carrera y un método estudiar.

Polimorfismo

class Profesor extends Persona {
    constructor(nombre, edad, materia) {
        super(nombre, edad);
        this.materia = materia;
    }

    saludar() {
        return `Hola, soy el profesor ${this.nombre} y enseño ${this.materia}.`;
    }
}

// Crear una instancia de la clase Profesor
const profesor1 = new Profesor('Carlos', 40, 'Matemáticas');
console.log(profesor1.saludar());

En este ejemplo, la clase Profesor sobrescribe el método saludar de la clase Persona para proporcionar una implementación específica para profesores.
