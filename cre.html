Introducción al Desarrollo de Páginas Web

El desarrollo de páginas web es el proceso de crear y mantener sitios web. Involucra varias tecnologías como HTML, CSS y JavaScript.

Ejemplo de HTML:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi Primer Sitio Web</title>
</head>
<body>
  <h1>¡Hola, Mundo!</h1>
  <p>Este es mi primer sitio web.</p>
</body>
</html>

Ejemplo de CSS:

body {
  font-family: Arial, sans-serif;
  background-color: #f0f0f0;
  margin: 0;
  padding: 20px;
}
h1 {
  color: #333;
  text-align: center;
}

Estos son solo ejemplos básicos para comenzar. Con el tiempo, aprenderás a crear diseños más complejos y funcionales utilizando estas tecnologías.
