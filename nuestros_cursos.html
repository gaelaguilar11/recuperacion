Cursos Disponibles

Introducción a la Programación con Python

Duración: 6 semanas

Descripción: Aprende los fundamentos de la programación utilizando Python. Este curso cubre desde conceptos básicos hasta estructuras de datos y funciones.

Desarrollo Web con HTML, CSS y JavaScript

Duración: 8 semanas

Descripción: Domina las tecnologías fundamentales para crear sitios web. Aprende a estructurar contenido con HTML, diseñar con CSS y añadir interactividad con JavaScript.

Programación Orientada a Objetos con Java

Duración: 10 semanas

Descripción: Adquiere habilidades avanzadas en programación orientada a objetos utilizando Java. Este curso incluye prácticas de laboratorio y proyectos en equipo.

Desarrollo de Aplicaciones Móviles con Kotlin

Duración: 12 semanas

Descripción: Aprende a desarrollar aplicaciones móviles para Android utilizando Kotlin. Este curso cubre desde los fundamentos hasta la publicación de tu propia aplicación en la Play Store.

Introducción al Machine Learning con Python

Duración: 8 semanas

Descripción: Explora los conceptos básicos del Machine Learning y cómo implementarlos en Python. Aprende sobre regresión, clasificación, y clustering.

Desarrollo de Juegos con Unity

Duración: 12 semanas

Descripción: Aprende a crear tus propios videojuegos utilizando Unity. Este curso cubre desde los fundamentos de la física de juegos hasta la programación de inteligencia artificial.
