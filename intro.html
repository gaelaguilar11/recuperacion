¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Describe la estructura de una página web y se compone de una serie de elementos que indican cómo se debe mostrar el contenido en el navegador.

Estructura Básica de un Documento HTML

Ejemplo:

<!DOCTYPE html>
<html>
<head>
<title>Título de la Página</title>
</head>
<body>
<h1>Mi Primer Encabezado</h1>
<p>Mi primer párrafo.</p>
</body>
</html>

Este es un ejemplo básico de un documento HTML. Vamos a desglosarlo en sus componentes principales:

Elementos HTML

Los elementos HTML son los componentes básicos de un documento HTML. Cada elemento está representado por etiquetas, como <h1>, <p>, etc.

Encabezados

Los encabezados se utilizan para definir títulos y subtítulos en una página. HTML tiene seis niveles de encabezados, desde <h1> (el más importante) hasta <h6> (el menos importante).

Ejemplo:

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>
<h4>Encabezado 4</h4>
<h5>Encabezado 5</h5>
<h6>Encabezado 6</h6>

Párrafos

Los párrafos se definen con la etiqueta <p>. Los navegadores añaden un espacio antes y después de cada párrafo.

Ejemplo:

<p>Este es un párrafo.</p>
<p>Este es otro párrafo.</p>

Enlaces

Los enlaces se definen con la etiqueta <a>. Puedes hacer clic en un enlace para ir a otra página o ubicación.

Ejemplo:

<a href="https://www.example.com">Visita Example.com</a>

Imágenes

Las imágenes se definen con la etiqueta <img>. Debes especificar el atributo src para indicar la fuente de la imagen y el atributo alt para proporcionar un texto alternativo.

Ejemplo:

<img src="ruta/a/la/imagen.jpg" alt="Descripción de la imagen">
